## Command Description
### tatcli version -h

```bash
tatcli version

Usage:
  tatcli version [flags]

Aliases:
  version, v


Flags:
  -h, --help=false: help for version
      --versionNewLine=true: New line after version number. If true, display Version Engine too

Global Flags: see tatcli -h

```
